Ihre E-Mail-Signatur

Richten Sie Ihre Signatur in wenigen Sekunden ein — so wird sie unter jeder E-Mail angezeigt:

1. Auf „Signatur kopieren“ klicken.
2. In Outlook öffnen: Einstellungen → Konten → Signaturen (klassisches Outlook: Datei → Optionen → E-Mail → Signaturen).
3. Neue Signatur anlegen, in das große Feld klicken und einfügen (Strg+V bzw. Cmd+V).
4. Speichern — fertig.
✓ Kopiert! Jetzt in Outlook einfügen.
